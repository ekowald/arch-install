# Arch Linux Installation Script

This script will install Arch Linux with X11, i3-gaps, and a minimalistic yet work-ready environment. Before using the script, please ensure that you've modified the partitioning, formatting, and mounting sections according to your system's requirements. Additionally, replace the placeholders for hostname and username with your desired values.

## Usage

1. Boot your system with the latest Arch Linux installation media.
2. Connect to the internet, either via Ethernet or Wi-Fi.
3. Update the system clock using the `timedatectl set-ntp true` command.
4. Download the script using the `curl` or `wget` command.
5. Give the script executable permissions using the `chmod` command:
6. Run the script as root:

After the script finishes, the system will reboot into your newly installed Arch Linux environment.
